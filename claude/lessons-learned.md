### 詳細
- `Edit` ツールの `replace_all` で `sed -i '' ` → `sed -i''` を置換した際、末尾スペースまで消えて後続の `"s/...` と結合した。置換結果のプレビューを意識すべき。
- GitHub Actions ワークフローを1ジョブ→2ジョブに分割した際、元のフローでは不要だった「Push reverts」ステップが必要になった理由を実装時に説明すべきだった。ユーザーから「そもそも必要？」と質問されて初めて説明した。
