# Nix 学習リソース

## 日本語リソース

### 体系的に学べる書籍形式

- [Nix入門](https://zenn.dev/asa1984/books/nix-introduction) - Nixのコンセプトと仕組みを解説。「Nixって何？」からスタートできる
- [Nix入門: ハンズオン編](https://zenn.dev/asa1984/books/nix-hands-on) - 実際に手を動かしながら学ぶ。Nix言語基礎、Flake、devShell、stdenvなどを網羅

### 記事・チュートリアル

- [Nix初心者がZero to Nixをやってみた](https://zenn.dev/hanao/articles/140d8a1ce32830) - 実体験ベースの学習記録
- [コマンドライン上で簡単に試せる Nix の使い方](https://ryota2357.com/blog/2025/use-and-start-nix-as-cli-mma2025b/) - CLIでの実践的な使い方
- [Nixとはなにか、パッケージマネージャとしての使い方の紹介](https://qiita.com/junjihashimoto@github/items/54f8e25071b2800ee2ee) - Qiitaの入門記事
- [NixOS を使い始める](https://blog.anqou.net/2024/12/start-nixos/) - NixOSの使用方法

## 英語リソース

### 公式・準公式

- [nix.dev - First Steps](https://nix.dev/tutorials/first-steps/index.html) - 公式チュートリアル（ここから始めるのが推奨）
- [Nix language basics](https://nix.dev/tutorials/nix-language.html) - 言語機能の解説
- [Zero to Nix](https://zero-to-nix.com/) - Determinate Systems製。Flakes中心の現代的アプローチ
- [Learn Nix | Determinate Systems](https://docs.determinate.systems/guides/learn-nix/) - Determinate Systemsのガイド
- [NixOS Wiki - Resources](https://wiki.nixos.org/wiki/Nix_Language:_Learning_resources) - 学習リソース一覧
- [nixos.org/learn](https://nixos.org/learn/) - 公式学習ページ

### 深掘り用

- [Nix Pills](https://nixos.org/guides/nix-pills/) - Nix内部への深い理解を得るための連載
- [Nixpkgs Manual](https://nixos.org/manual/nixpkgs/stable/) - パッケージビルドの詳細ガイド
- [Nix Reference Manual](https://nixos.org/manual/nix/stable/) - コマンドリファレンス

### その他

- [Nix by example](https://nixbyexample.dev/) - 例で学ぶステップバイステップチュートリアル
- [The Nix Tour](https://nixcloud.io/tour/) - ブラウザ上でインタラクティブに学べる
- [awesome-nix](https://github.com/nix-community/awesome-nix) - コミュニティキュレーションのリソース集

## 推奨学習順序

1. **[Nix入門](https://zenn.dev/asa1984/books/nix-introduction)** で概念理解
2. **[Zero to Nix](https://zero-to-nix.com/)** または **[Nix入門: ハンズオン編](https://zenn.dev/asa1984/books/nix-hands-on)** で実践
3. **[nix.dev](https://nix.dev/)** で公式ドキュメントに慣れる
4. **[Nix Pills](https://nixos.org/guides/nix-pills/)** で内部理解を深める
